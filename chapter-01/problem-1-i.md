# Spivak Chapter 1, Problem 5(iii)

**Statement:** If $a < b$ and $c > d$, then $a - c < b - d$.

**Proof:**
Since $a < b$, we have $b - a > 0$.
Since $c > d$, we have $d - c < 0$.
Adding: $(b - a) + (d - c) > 0$
$\Rightarrow (b + d) - (a + c) > 0$
$\Rightarrow a - c < b - d$. ∎
